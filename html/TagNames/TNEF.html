TNEF Tags

Information extracted from Transport Neutral Encapsulation Format (TNEF) files (eg. winmail.dat). But note that the exiftool application doesn't process files with a .DAT extension by default when a directory name is given, so in this case either specify the .DAT file(s) by name or add -ext+ dat to the command.
